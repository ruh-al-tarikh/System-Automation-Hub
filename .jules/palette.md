## 2025-05-14 - [Enhanced Webhook Visibility]
**Learning:** Terminal-based automation tools benefit significantly from visual hierarchy and source identification. Distinguishing between generic web requests and specific platform events (like GitHub webhooks) via icons and colors reduces cognitive load for developers monitoring logs. Pretty-printing JSON in the console is a simple but high-impact accessibility win for readability.

**Action:** Always check if JSON output can be pretty-printed and if event sources can be visually distinguished in CLI/log outputs.
